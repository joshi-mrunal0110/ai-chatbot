## ✅ **ATS-Friendly Project Description — AI Chatbot (HTML, CSS, JavaScript, Gemini API)**

**AI Chatbot using Gemini API | HTML, CSS, JavaScript**
**Tech Stack:** HTML, CSS, JavaScript, Gemini API, REST APIs

**Description:**
Developed an interactive AI-powered chatbot using the Google Gemini API with a fully responsive front-end built using HTML, CSS, and JavaScript. Implemented asynchronous API communication to generate dynamic, human-like responses in real time. Integrated user-friendly UI components, message rendering, loading animations, and error handling to ensure smooth interaction.

**Key Features:**

* Integrated **Google Gemini API** for natural language processing and AI-generated responses
* Designed **responsive UI** with clean layout using HTML & CSS
* Implemented **async/await** based API calls using JavaScript
* Created **chat message rendering**, typing indicators, and scroll-to-bottom features
* Added **input validation**, API error handling, and fallback response logic
* Ensured seamless user experience with lightweight front-end architecture
* Optimized for browser performance using vanilla JavaScript

**Responsibilities:**

* Built front-end interface using HTML & CSS
* Developed JavaScript logic for sending/receiving AI responses
* Connected the chatbot to Gemini API using fetch() and REST endpoints
* Tested and optimized UI interactions and browser compatibility

**Outcome / Impact:**

* Achieved **real-time AI response generation** with minimal latency
* Improved user interaction with smooth UI/UX
* Enhanced understanding of API integration, JavaScript DOM manipulation, and AI workflows
